# Platform Component

This is an abstraction layer over externals chat systems like Slack, Gitter, Vk, Facebook, etc.

## Requirements

- php >= 7.1

## Installation

1. Install composer: https://getcomposer.org/download/
2. Run `composer require karma/platform`
